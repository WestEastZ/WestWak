## 5. ✨ 주요 기능

- 사용자 인증 및 보안
- 게시물 CRUD
- 차트를 활용한 음원 정보 제공
- 최신 Youtube 영상 / 실시간 방송 정보
- AI 기반 얼굴 유사도 분석 시스템
